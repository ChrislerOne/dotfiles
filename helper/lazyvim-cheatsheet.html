☁ AeroSpace — Window Manager
FOCUS
Move focus between windows
⌘⌥hFocus left
⌘⌥jFocus down
⌘⌥kFocus up
⌘⌥lFocus right
MOVE
Move windows around
⌘⌥⇧hMove left
⌘⌥⇧jMove down
⌘⌥⇧kMove up
⌘⌥⇧lMove right
WORKSPACES
Switch & move to workspaces
⌘⌥1-5Switch workspace
⌘⌥⇧1-5Move window to workspace
LAYOUT
Window layout & misc
⌘⌥fFullscreen
⌘⌥⇧SpaceFloat / tile toggle
⌘⌥- / =Resize
⌘⌥⇧rReload config

SPLITS

📡 tmux — Terminal Multiplexer Prefix = Ctrl+b
SESSIONS
Create & manage sessions
Ctrlb dDetach session
Ctrlb sList sessions
Ctrlb $Rename session
WINDOWS
Create & switch windows
Ctrlb cNew window
Ctrlb n / pNext / Previous window
Ctrlb 0-9Go to window by number
Ctrlb ,Rename window
Ctrlb &Close window
PANES
Split & navigate panes
Ctrlb vSplit vertical
Ctrlb -Split horizontal
Ctrlb hjklNavigate panes (vim-style)
Ctrlb HJKLResize panes
Ctrlb xClose pane
Ctrlb zZoom pane (toggle)
COPY MODE
Vi-style copy & scroll
Ctrlb ⌥5[Enter copy mode
v (in copy mode)Begin selection
y (in copy mode)Yank selection
qExit copy mode
🎓 Neovim Basics — Modes
NORMAL
Default mode — navigate & run commands
hjklLeft / Down / Up / Right
w / bNext / Previous word
0 / $Start / End of line
gg / GStart / End of file
ddDelete line
yyYank (copy) line
pPaste
u / CtrlrUndo / Redo
.Repeat last action
⇧7/termSearch ( /term )
n / NNext / Previous match
INSERT
Type text — like a normal editor
iInsert before cursor
aInsert after cursor (append)
IInsert at start of line
AInsert at end of line
o / ONew line below / above
Esc or jkBack to Normal
VISUAL
Select text — then apply an action
vVisual (character-wise)
VVisual Line (line-wise)
CtrlvVisual Block (column-wise)
dDelete selection
yYank (copy) selection
⇧. / ⇧,> / <Indent / Outdent ( > / < )
EscBack to Normal
COMMAND
Ex commands — save, quit, etc.
:wSave
:qQuit
:wqSave & Quit
:q!Quit without saving
:%s/old/new/gFind & Replace (file)
:42Jump to line number
